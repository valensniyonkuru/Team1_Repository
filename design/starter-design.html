React Component Mapping

Suggested component names and what they correspond to in this design.

<NavBar />Top navigation bar with logo, search, new post button, and user avatar
<FilterBar />Horizontal pill-style category filter tabs
<PostCard />Individual post preview card in the feed grid
<PostDetail />Full post view with complete content and metadata
<CommentSection />Comment list with input box and individual comment items
<CommentItem />Single comment with avatar, author, date, and text
<CreatePostForm />Post creation form with title, category, and rich text content
<DashboardCard />Single metric card showing label, value, and trend
<BarChart />Category distribution bar chart visualization
<LineChart />Weekly activity trend line chart visualization
<Leaderboard />Top contributors table with rank, name, activity bar, count
<Badge />Color-coded category badge (Events, Lost & Found, etc.)
<AuthCard />Centered authentication card used for Login and Register pages
<Button />Reusable button with variants: primary, outline, success, sizes: sm, md, lg
<FormInput />Styled input with label, supports text, email, password, select, textarea
