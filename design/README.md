# CommunityBoard Starter Design

This folder contains a starter design for the CommunityBoard application.

## Files
- `starter-design.html` - Open this file in a browser to see all screens and the design system

## For Frontend Developers
Use this design as your starting point for React component implementation. You do NOT need to wait for the UI/UX designer's wireframes to begin coding.

## For UI/UX Designers
This is a baseline design. Your role is to:
1. Improve and refine this design (colors, typography, spacing, interactions)
2. Create your own Figma prototype with enhanced UX
3. Deliver your complete design system deliverables
4. Communicate design updates to the frontend developer incrementally
